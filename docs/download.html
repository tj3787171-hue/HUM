What's on the ISO

hum-snap-bypass.sh

Extract, FUSE-mount, inspect, and run snap packages without snapd. No root required.

hum-snap-server.sh

Loop-mount snaps at /snap via kernel squashfs + cgroup v2 scope. Replaces snapd mount units.

hum-dev-netns.sh

Create network namespaces, veth pairs, and dummy interfaces for LAN dev naming.

deepseek_db_link.py

Index DeepSeek backup exports into a local SQLite database. Zero pip dependencies.

Quick start after download

# Mount the ISO
sudo mkdir -p /mnt/hum
sudo mount -o loop hum-toolkit.iso /mnt/hum

# Copy scripts to your system
cp -r /mnt/hum/hum/scripts/ ~/hum-toolkit/

# Or run directly from the mount
bash /mnt/hum/hum/scripts/hum-snap-bypass.sh deps
bash /mnt/hum/hum/scripts/hum-snap-bypass.sh info some-package.snap
sudo bash /mnt/hum/hum/scripts/hum-snap-server.sh up

# Unmount when done
sudo umount /mnt/hum

Build your own ISO

Clone the repo and run the build script to generate a fresh ISO from source: 

git clone https://github.com/tj3787171-hue/HUM.git
cd HUM
bash scripts/hum-build-iso.sh dist/hum-toolkit.iso

Requires: genisoimage, syslinux-utils, isolinux
